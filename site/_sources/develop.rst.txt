ASK Amy development
==================

ASK Amy is an open source project and we value contributions!
If you would like to contribute to the project with code, tutorials,
sample apps or documentation please reach out
to `Dan Higgins <mailto:dan.higgins@ilandapps.com>`_.

* The code can be found in a Git repository, at
  https://github.com/dphiggs01/ask_amy.
* Issues and feature requests should be raised in the `tracker
  <https://github.com/dphiggs01/ask_amy/issues>`_.
